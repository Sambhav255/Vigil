# Vigil

Geopolitical event severity intelligence dashboard.

## App (Next.js)

The production app lives in [`flashpoint-app/`](./flashpoint-app/). From that directory:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Standalone prototype

[`flashpoint-dashboard.jsx`](./flashpoint-dashboard.jsx) is a self-contained React prototype (reference only).
